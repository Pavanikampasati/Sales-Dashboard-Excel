 📊 Sales Dashboard (Excel Project)

 Overview

This project is an interactive Sales Dashboard created using Microsoft Excel.

 🔧 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Graphs
* Slicers

 📊 Features

* Sales by Category, Region, Segment
* Sales Trends Over Time
* Shipping Analysis
* Interactive Filters (Slicers)

 📁 File Structure

* Raw_Data → Original dataset
* Cleaned_Data → Processed data
* Analysis → Pivot tables
* Dashboard → Final visualization

 Purpose

To analyze sales performance and present insights using an interactive dashboard.
